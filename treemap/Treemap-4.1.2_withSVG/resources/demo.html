Demo Files

nba-no-hierarchy.tm3 - Statistics of NBA basketball players
  • 424 professional basketball players with statistical information (i.e. points per game)
  • Hierarchy - No default hierarchy is specified.
nba-with-hierarchy.tm3 - Statistics of NBA basketball players
  • 424 professional basketball players with statistical information (i.e. points per game)
  • Hierarchy - NBA, conferences, divisions, teams, players
nba-settings.tms - Example settings file for the NBA data
  • Size of the rectangle is proportional to the points earned by each player.
  • Color shade is proportional to the technical fouls made by each player.
  • Hierarchy - NBA, divisions, teams, players
census.tm3 - Census information of Maryland Industries
  • Four years of information (93-96) regarding number of establishments, employees, and payroll for different industries
  • Hierarchy - Maryland, counties, industries, years
election-with-hierarchy.tm3 - Map of the United States with election information
  • Election information such as number of electoral votes, state population, and winning party for previous 3 elections
  • Hierarchy - United States, region, state
43causesofdeath-65plus.tm3 - Statistical data of number of deaths per 100,000 people over 65 years of age.
  • Information such cause of death, number of deaths in 1981 and 1998, and the percent change in deaths since 1981 provided.
  • Hierarchy - Category of disease, cause of death.
43causesofdeath-65plus.tms - Example settings file for 43causesofdeath-65plus.tm3 data
  • Size of the rectangle is proportional to the "1998 Deaths per 100,000".
  • Color shade is proportional to the "%change since 81"
  • Hierarchy - Default hierarchy (Category of disease, cause of death)
Firearms.tm3 - Information on males and females involved in accidents due to firearms.
  • Information such as gender, race, age, and cause of death specified.
  • Hierarchy - No default hierarchy specified.
ProjectReport_no_hierarchy.tm3 - Information on some technical projects.
  • Information such as project name, status, percent complete, budget allocated, budget balance, current cost, scheduled finish etc.
  • Hierarchy - No default hierarchy specified.
ProjectReport_with_hierarchy.tm3 - Information on some technical projects.
  • Information such as project name, status, percent complete, budget allocated, budget balance, current cost, scheduled finish etc.
  • Hierarchy - Region, Location and project name
ProjectReport_no_hierarchy.tms - An example settings file for ProjectReport data.
  • Size of the rectangle is proportional to the budget allocated for the projects
  • Color is porportional to the budget balance. For example, red rectangles represent the -ve budget balance.
  • Hierarchy - Budget balance, project status
